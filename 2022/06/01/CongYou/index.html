当过 \(04年-08年,12年-18年\) 班主任，招聘实习生

信息所关注信息技术，发展迅速，在各个行业应用广泛， \[ \large \begin{cases}数据处理:收集、管理、检索数据(高性能)\\数据分析:了解存储数据之间的分布、关联、信息\end{cases} \]

\[ \large 数据分析\begin{cases}数据挖掘:预测(经济)、医疗(身体状况信息)\\数据分类:(类似图书管理系统)机器自动分类庞大数据\\数据关联:(智能用户画像)推荐系统,异常检测\end{cases} \]

监督、指导过程，当前演化为无指导化（不需要原始数据集，聚类，\(\mbox{Class}\) ）

应用：互联网（给定关键词检索，相似的内容自动分好组）

实验室工作，结构化的时序，文本，网络结构三类数据

  • 时序数据：例如（手机）传感器判断场景、监测健康信息(华为、西门子研究院、医疗团队)

  • 文本数据：自然语言处理（ \(\mbox{NLP}\) ）（目前存在瓶颈，语义分析较浅，主要运用统计分析的方法统计频率），话题识别，文本相似性便于分类，正负情感分析）

  • 网络结构数据：和图相似（超链接相连），利用节点数量寻找最关键信息，模块化思想引入多个类进行分析，程序容易有局限性，判断节点是否出现失效

    未来数据的融合程度更高。

多层神经网络节点多，参数多，需要大量的参数，会引发“过拟合”（模型过于依赖所给样例）

实验室方向欢迎大家来，本科生、研究生一起组会。只要感兴趣，有精力，就可以来了解学习。

算力需要提升

原始架构在传递方面有一定局限，\(1000\mbox{GB/s}\)，但是最终数据流不到 \(\mbox{1GB/s}\) \[ \begin{array}{|c|c|c|c|c|} \hline & \text { 计算器件 } & \text { 计算范式 } & \text { 算子 } & \text { 架构 } \\ \hline \text { 冯诺依曼计算系统 } & \text { 场效应晶体管 } & \text { 布尔逻辑数字计算 } & \text { 与门, 非门 } & \text { 存算分离 } \\ \hline \text { 存算一体计算系统 } & \text { 忆阻器 } & \text { 物理定律模拟计算 } & \text { 乘法, 加法 } & \text { 存算一体 } \\ \hline \end{array} \]

基于忆阻器的存算一体技术

模拟计算机会导致许多误差（在物理层面的），虽然其在矩阵乘法效率上有很大的优势，但是有一定局限性。

  • 将带来从底层器件到编译器等层面的改变, 实 现新计算机系统, 但不改变现有编程语言。 > 新计算系统能效将提高 \(10^{2}-10^{3}\) 倍以上, 达到 1POPs/W。 > 单芯片算力可以得到有效提升, 达到 500TOPs 或者 1POPS 。

大脑对少于一定值的信号会忽略，但对大于一定值的信号会放大，类脑计算能在一些复杂问题中发挥作用（入）

大脑神经元 \(10^{11}\) 量级，突触为 \(10^{15}\) 量级，要实现仿生型的芯片，还有很多关键技术需要突破

小结

  • 后摩尔时代传统架构AI芯片面临严峻的算力和能效瓶颈；
  • 存算一体是突破传统架构中存储墙、功耗墙的好办法；
  • 基于存算一体(模拟计算)的 \(\mathbf{NPU}\) 任重道远:新器件、新电路、新架构、新算法、新工具链；
  • 更远的未来,需要超越模拟计算,开发"类脑芯片"?

光计算使用光的干涉来模拟 \(0\) 和 \(1\)，还缺乏功能演示，没有到 \(\mathbf{array}\) 的环节
